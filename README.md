# Alliatus - Mastermind Landing Page

A fully responsive, high-converting landing page implementation based on the popular [Alliatus Figma Community Template](https://figma.com). This project converts the original AI-assisted UX Pilot design into clean, production-ready code tailored for mastermind groups, coaching cohorts, and premium communities.

## 🚀 Live Demo
https://alliatu.netlify.app/

## ✨ Features
- **Figma-Perfect Accuracy:** Matches the typography, spacing, and elegant aesthetic of the original Alliatus design.
- **Conversion-Optimized Layout:** Tailored specifically for entrepreneurs, creatives, and solopreneurs to join community programs.
- **Fully Responsive:** Seamless viewing experience across mobile, tablet, and desktop devices.
- **Clean Architecture:** Well-structured components making text, color, and image swaps effortless.

## 🛠️ Tech Stack
*   **Design Source:** Figma (Alliatus Template)
*   **Frontend:** [HTML5, CSS3, Javascript]
*   **Deployment:** [Netlify]

## 🔧 Getting Started

1. Clone the repository:
   ```bash
   git clone https://git@github.com:nofugazidev/Alliatu.git
   ```

   ```bash

   ```
